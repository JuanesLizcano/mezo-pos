# mezo 🍽️
> Sistema de punto de venta para cafeterías y restaurantes en Colombia

## ¿Qué es mezo?
SaaS POS con IA para negocios colombianos.
Onboarding en 5 minutos, mesas en tiempo real, reportes inteligentes.

## Stack
React · Firebase · Tailwind CSS · Claude API · DALL·E 3

## Estado
En desarrollo activo — Beta con primer cliente Mayo 2026

## Cómo correr el proyecto localmente
1. Clonar: git clone https://github.com/JuanesLizcano/mezo-pos.git
2. Instalar: npm install --legacy-peer-deps
3. Crear archivo .env con las variables de Firebase (pedir al equipo)
4. Correr: npm start
